User Guide

The aim of 147 pal is to be as intuitive as possible. However, for those who are not tech-savvy, here's a guide for you!

Main Menu

This is the main menu. From here you can navigate to the key parts of the app. The key parts of the app are:

  • New Game
  • Rules
  • Profiles
  • Practice Routines
  • User Guide

New Game

This is the new game screen. From here you can choose how many frames are going to be played and the players that will participate in the game using the dropdown menus. This will allow for the stats to be updated on the user profiles when the game has finished. An option is also available to change how many snookers are required before the frame ends. At the bottom of the screen there are buttons that allow you to either go back to the main menu or create the game with the settings chosen.

In Game (on red)

This is the in-game screen. At the top of the screen you can see the names of the players, with an arrow indicating who's turn it is. The scores of the respective players are in the same box as their name. The middle box containes the total number of frames in the game and the number of frames each player has won. Down the centre of the screen the elapsed time, points behind or ahead, points remaining, snookers required and current break are visible. As it is the start of a break, only the red ball is visible (since this is the only ball that is legal to pot at the current time).
At this stage the active player will take their shot. There are 3 possible outcomes from their shot

  • If they pot a red then the red ball button should be pressed. The colour balls will then become available to pot and the on-screen stats will be updated accordingly.
  • If they foul then the foul button should be pressed. This will take the user to the foul screen and the committed foul can be processed. After completion on-screen stats will be updated and the turn will be swapped.
  • If they miss then the miss button should be pressed. This will swap the turn to the other player.

A button is also available to end the game prematurely and progress to the game stats screen.

In Game (on colour)

This is the in-game screen when the player has potted a red. As you can see the coloured balls have now appeared, meaning they are pottable. The same stats appear on the screen here as they did before. You can now see the stats have been updated on the screen accordingly.

Add Foul

Here is the add foul screen. This appears when the "add foul" button is pressed in game. From here the ball that the foul was committed on should be pressed. For example if the player was aiming for black and missed it, or potted the white, the black ball should be pressed. This will then add the required points to the opponent, end the current break and return to the in-game screen.

Frame Stats

This is the frame stats screen. From here you can see the stats from the frame when it has finished. If it is the final frame of the game (as shown on screen) the "game stats" button will be shown, allowing the user to continue to the game stats screen. If there are more frames to be played then the "next" button will be shown, allowing the user to continue to the next frame.

Game Stats

This is the game stats screen. From here you can see the stats from the completed game and return to the menu. The stats on the screen will be uploaded to the database so they are stored for each profile.

Profiles

This is the profiles menu. From here you can see the profiles that have been created on the device. This app can be used with or without internet so it is possible to have multiple local profiles without having any internet connection. Clicking on one of the profiles will navigate you to the stats of that profile as shown below. Clicking the "create profile" button will navigate you to the create profile screen.

Profile Stats

This is the profile stats screen. On this screen you can see the personal stats for the specified profile. Pressing the "delete profile" button will remove the profile from the local database and the online database.

Create Profile

This is the create profile screen. From here you can type in the name of the profile you would like to create and then add the profile to the local and online database by pressing "create profile"

View Routines

This is the view routines screen. From here you can see the list of preset routines that are availalbe to play. Clicking on a routine you would like to play will take you to the view routine page of that routine.

View Routine

This is the view routine screen. From here you can see a larger version of the routine and set the balls up in the correct positions. To play the routine you must choose the player profile you would like to use from the dropdown menu and then press "start". To view the leaderboard for the current routine, click the "leaderboard" button.

Leaderboard

This is the leaderboard screen. From here you can see the best scores for the current routine. The rows in the table are ordered by score from highest to lowest.

In Routine

This is the in-routine screen. From here you can see the current stats of the routine. To play through the routine you must press the buttons of the colours that you pot.

Routine Stats

This is the routine stats screen. From here you can see the stats from the finished routine.
